# 📚 Exercícios em Python

Este repositório contém exercícios e atividades desenvolvidas em **Python** durante meus estudos de programação.

## 🎯 Objetivo

Praticar lógica de programação, entrada e saída de dados, estruturas condicionais e operações matemáticas utilizando Python.

## 📂 Conteúdo da pasta

* **apresentacao_usuario.py** → Programa que pede nome e idade do usuário
* **operacoes_basicas.py** → Calculadora com soma, subtração, multiplicação e divisão
* **verificar_turno.py** → Programa que identifica o turno de estudo (M, V, N)
* **menor_preco.py** → Código que encontra o menor preço entre três valores

## 🧠 Conceitos praticados

* Entrada de dados (`input`)
* Saída de dados (`print`)
* Condicionais (`if`, `elif`, `else`)
* Funções (`def`)
* Operações matemáticas
